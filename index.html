📸 Как добавить фото через Flickr:
Для галереи на сайте:
  1. Откройте фото на Flickr
  2. Нажмите Правой кнопкой → "Копировать адрес изображения"
  3. Вставьте ссылки ниже через запятую (3-5 лучших фото)
Для полного альбома:
  1. Откройте альбом на Flickr
  2. Скопируйте ссылку из адресной строки
  3. Вставьте в поле "Ссылка на альбом"
3-5 лучших фото для отображения на сайте. Ссылки должны заканчиваться на .jpg
Все фото из поездки. На сайте появится кнопка "Смотреть все фото".
